############
Installation
############

``dohq-teamcity`` is compatible with Python 2.7 and 3.4+.

Use :command:`pip` to install the latest stable version of ``dohq-teamcity``:

.. code-block:: console

   $ sudo pip install --upgrade dohq-teamcity

The current development version is available on `github
<https://github.com/dohq-teamcity/dohq-teamcity>`__. Use :command:`git` and
:command:`python setup.py` to install it:

.. code-block:: console

   $ git clone https://github.com/dohq-teamcity/dohq-teamcity
   $ cd dohq-teamcity
   $ sudo python setup.py install
